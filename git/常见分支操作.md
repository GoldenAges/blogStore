## 删除分支操作
### 删除本地分支 
        git branch -D branchName
### 删除远程分支
        git push origin --delete branchName
或者
        
        git push origin :branchName

当出现删除错误

    unable to push to unqualified destination:

运行命令：

    git fetch -p origin

原因：本地目录信息没有同步远程，可能远程分支已经被删除
